## Charter

M7 built the TEI gate as a metadata oracle: `output_format == "xmltei"` forces
`with_metadata=True` (settings.py:144-149), so the gate is the only one exercising the full
metadata-extraction subsystem. Today it **masks the entire `<teiHeader>`** because mdrcel's
metadata is the weaker side. **M8 = drive mdrcel's metadata subsystem to byte-parity with Python
and unmask the `<teiHeader>` incrementally — field by field — keeping every gate green.**

### Scope (BRIEF §3)
- **In:** dates (htmldate parity), hostname/URL (courlan parity), author/sitename quality,
  `filedate` (no slot today), and any other header field that diverges (categories/tags/license/
  description).
- **Out (keep masked/deferred):** blake2b fingerprint (`<note type="fingerprint">`), U+2063 /
  `remove_control_characters`, the other four gates' masks, comments/dedup/precision-recall/lang.

### Anti-inversion doctrine (non-negotiable)
ADRs/journals are starting points, not recipes. Every root-cause claim verified against vendored
Python source:
- `benchmark/oracles/trafilatura/.venv/Lib/site-packages/trafilatura/metadata.py`
- `…/courlan/__init__.py` (hostname/URL)
- `…/htmldate/core.py` (date discovery + parsing)

### Triage discipline (BRIEF §4, 3-bucket)
1. **Substantive:** mdrcel fixed, byte-equal. No ADR.
2. **`PYTHON_UNDER_EXTRACT_ALLOWLIST`:** Python wrong/non-spec; ADR in `wrk_docs/m8-allowlist/`.
   High bar — prefer matching Python (fidelity = bugs and all) unless Python crashes / malformed XML.
3. **`DEFERRED_KNOWN_DEFECT`:** mdrcel wrong, fix blocked (dependency); ADR in `wrk_docs/m8-deferred/`.

GREEN = no untriaged fixtures + accounting invariant holds.

### Exit criteria (BRIEF §8)
`<teiHeader>` unmasked except blake2b fingerprint + any U+2063-only divergence; all gates green;
ADRs for every allowlist/deferred; journal; Cargo version bumped (MINOR if `filedate` public field added).

Verified against vendored source: `extract_domain(url, fast=True)` (courlan/urlutils.py:14-62) returns

Plan: fix B/C/D/G/H (general rules) first, then narrow the gate mask from whole-`<teiHeader>` to
field-granular (fingerprint-only), then triage E (date discovery) + F (author) residuals.

### Stage 1 — general-rule metadata fixes (filedate / hostname / date / sitename) (2026-05-22)

Five source changes (all in `src/trafilatura/metadata*.rs` + `output.rs`), each a faithful port of a
verified Python rule:

1. **filedate slot** (Class B). Added `Metadata.filedate`; `extract_metadata` sets it to today
   (`%Y-%m-%d`) via a new `today_utc()` (SystemTime + Howard Hinnant `civil_from_days`, no new dep).
   Rendered in `<date type="download">` (output.rs). Mirrors `metadata.py:586` + `settings.py:202`.
   **Caveat:** Python uses *local* `datetime.now()`; mdrcel uses UTC. On this UTC+1 host the civil
   dates agree except in the ~1h local-midnight window — accepted residual vs a timezone dep.
2. **Date config + ordering** (Classes C+E). mdrcel had `metadata.date = extract_date(...)` gated on
   `is_none()`, so the raw JSON-LD `datePublished` (`2002-06-06T01:53:27Z`) won and htmldate never
   ran. Python (metadata.py:547) assigns `find_date(...)` UNCONDITIONALLY. Fixed: call `extract_date`
   unconditionally; and fixed `extract_date`'s config to Python's `set_date_params` — `original_date=
   True` (was False), `max_date=today` (was 9999 sentinel). htmldate's `%Y-%m-%d` output also fixed
   the truncation (Class C). NB: the htmldate-parity gate builds its OWN Extractor (original=False,
   max=9999) so it is unaffected.
3. **extract_domain** (Class D). The doc-comment claimed Python returns the full host — WRONG.
   Reimplemented as DOMAIN_REGEX group-1 (rightmost non-final label ≥4 chars to end) + a `www`-strip
   approximation of the `get_tld` PSL fallback (the `tld` PSL is not vendored; the approximation
   covers every corpus fallback: gov.uk/sec.gov/bbc.com/w3.org). Added 3 unit tests vs the live oracle.
4. **sitename normalisation + URL fallback** (Class G). Ported `metadata.py:559-572`: lstrip `@`
   (`@rustlang`→`rustlang`), `python_title_case` if dot-less + lower-initial (`rustlang`→`Rustlang`),
   else `META_URL` URL-host fallback (`meta_url_sitename`). Reordered sitename after url/date per Python.

**Result:** corpus header census dropped from ~35 diverging fixtures to **16** (excl. always-masked
fingerprint). Remaining: author quality (~9), date-discovery residual (3: 39ca4af9, 8d5cc524,
9c64e8e3), URL %-encoding (2: 5714710c, f76ec833). `cargo test --lib` 1161 green (updated the one
`extract_domain_simple` test that encoded the old wrong behaviour); clippy clean.
